Это MVP-платформа для пригонщиков авто.

Цель MVP:
Дать пригонщику инструмент для ведения клиентов и показа им подходящих автомобилей,
без автоматизации и без масштабирования.

Роли:
- Платформа
- Пригонщик
- Клиент пригонщика
- Менеджер платформы (ручная обработка)

Основные сущности:

Partner:
- id
- name
- slug
- contacts_json

Client:
- id
- partner_id
- name

SearchCriteria:
- id
- client_id
- country (Korea | China | Europe)
- source_site
- search_url

Car:
- id
- client_id
- title
- description
- images_json
- source_url
- created_at

Функционал MVP:

1. Главная страница сайта без брендинга и контактов.
2. Публичные страницы партнёров: /p/:slug
3. Публичные страницы клиентов партнёров: /p/:slug/c/:clientId
4. Закрытая админка для партнёров (логин/пароль, создаются вручную).
5. В админке:
   - создание клиентов
   - добавление критериев поиска
6. Парсер:
   - периодически проверяет search_url
   - если появляется новая ссылка — сохраняет её
7. Менеджер вручную добавляет карточку авто в БД.
8. Карточки авто отображаются на странице клиента.

Ограничения:
- без регистрации
- без оплаты
- без ролей
- без автоматической загрузки авто
- всё максимально просто
